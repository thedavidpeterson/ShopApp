This is a shopapp I created to test stores, redux, redux thunk and working with firebase.

Features:
- Create new account
- Login
- View Products, 
    - Select a product (read more info)
    - Add to cart 
    - Product:
        - Name
        - Price 
        - Image 
        - Description
- View Cart 
    - Summary of items 
    - Total cost 
    - Place order  
View Orders,
    - Orders made
Admin, 
    - Add more products to the store 
    - Edit current products 

Logout
